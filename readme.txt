# cv20222

vanilla-style-transfer.ipynb: implementation of 'A Neural Algorithm of Artistic Style'
fast-style-transfer.ipynb: implementation of 'Perceptual Losses for Real-Time Style Transfer'
style: folder containing models produced by fast-style-transfer.ipynb
style-transfer-inference.ipynb: evaluate the result of 2 models

Requirement: Install streamlit if you want to run in your local host

How to run the code:
Way 1: Enter command line in terminal "streamlit run main.py"
Way 2: According to the link: https://cv20222-1-rdopile8jk8.streamlit.app/
